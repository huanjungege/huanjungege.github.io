History模式

vue-router默认是hash模式，使用这种模式后端不需要额外的处理，前端路由主要通过URL中的hash(#号)来实现不同页面之间的切换，并不会向后端发送请求，并且HTTP请求中不会包含hash相关的内容，

但是使用默认模式会url会出现一个#号，比较丑，所以可以使用history模式，这种模式充分利用 history.pushState API 来完成 URL 跳转而无须重新加载页面。
