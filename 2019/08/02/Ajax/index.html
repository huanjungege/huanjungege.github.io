AJAX

Ajax = Asynchronous JavaScript and XML（异步的JavaScript和XML）

在浏览器中，我们可以在不刷新页面的情况下，通过ajax的方式去获取一些新的内容

发送Ajax请求的五个步骤

  • 创建异步对象。即XMLHttpRequest对象。
  • 使用open方法设置请求的参数。open(method,url,async)
  • 发送请求
  • 注册事件。注册onreadystatechange事件，状态改变时就会调用。
